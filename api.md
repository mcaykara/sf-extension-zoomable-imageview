## ZoomableImageView
**Kind**: global class  
**See**: http://ref.smartface.io/#!/api/UI.ImageView

ZoomableImageView is extention of [ImageView](http://ref.smartface.io/#!/api/UI.ImageView). This view component 
provides zooming ability based on gestures.  
**Since**: 1.0
